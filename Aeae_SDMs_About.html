Ecological Niche Modeling for Aedes aegypti

Climate-driven species distribution analysis in Haiti

About This Project

This repository hosts an ecological niche modeling analysis for Aedes aegypti mosquitoes in Haiti, utilizing ensemble Boosted Regression Tree (BRT) models to understand current species distribution patterns and predict changes under multiple climate warming scenarios.

Key Focus: Assessing how climate change may alter habitat suitability for this disease vector, which transmits dengue, Zika, chikungunya, and yellow fever.

Methodology

Statistical Approach

Ensemble Modeling 30 BRT model replicates with pseudo-absence generation
Climate Data 19 bioclimatic variables from WorldClim
Future Projections CMIP6 scenarios at 1.5°C, 2°C, 3°C, and 4°C warming
Visualization Interactive Leaflet maps with plasma color scheme

Technologies Used

  • R Programming: Core analysis platform
  • Key Packages: dismo, gbm, raster, leaflet, sf
  • Visualization: ggplot2, viridis (plasma palette), RColorBrewer
  • Climate Data: geodata for CMIP6 warming level ensembles
  • Web Platform: Quarto documents rendered to HTML, hosted on GitHub Pages

Outputs & Visualizations

Interactive Maps

Explore dynamic habitat suitability predictions with toggleable layers:

  • Baseline ensemble suitability (current climate)
  • Individual BRT model predictions (30 replicates)
  • Warming level projections (1.5°C through 4.0°C)
  • Model uncertainty metrics (CV, standard deviation)
  • Population density and IDP distribution overlays

Static Visualizations

High-resolution figures using the plasma color scheme for publication-ready graphics showing spatial patterns of habitat suitability and prediction uncertainty across Haiti.

Data Sources

  • Occurrence Data: Field trap surveys across Haiti
  • Climate Data: WorldClim 2.1 (0.5 arc-minute resolution)
  • Future Climate: CMIP6 multi-model ensemble projections
  • Population: Gridded Population of the World (GPW v4.11, 2020)
  • Administrative Boundaries: Haiti CNIGS ADM3 level shapefiles
  • IDP Data: Internally displaced persons assessment data
